picfun.particle — Type

particle represents the phase-space coordinates of a particle.

Fields

  • pos: Position in space
  • vel: Velocity
source
picfun.x — Method
x(p::particle)

Accesses the particle's position. This is wrapped in a function so that we can define the adjoint as

$\bar{x} ightarrow \mathrm{particle}\left( \bar{x}, 0 \right)$.

source
picfun.v — Method
v(p::particle)

Accesses the particle's velocity. This is wrapped in a function so that we can define the adjoint

$\bar{v} ightarrow \mathrm{particle}\left( 0, \bar{v} \right)$.

source
